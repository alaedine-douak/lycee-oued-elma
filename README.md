# ثانوية الإخوة الشهداء مراح - موقع حديث وسريع الاستجابة

## نظرة عامة

موقع ويب حديث وسريع الاستجابة لثانوية الإخوة الشهداء مراح في وادي الماء، مصمم بأفضل الممارسات في تصميم الويب.

## الميزات الجديدة

### 🎨 التصميم الحديث

- **نظام ألوان احترافي**: ألوان أساسية وثانوية منسقة بعناية
- **تدرجات جميلة**: تدرجات لونية في الأقسام الرئيسية
- **ظلال وتأثيرات**: استخدام محترف للظلال لإضافة العمق
- **رموز تعبيرية**: أيقونات سهلة الفهم بدلاً من الصور المعقدة

### 📱 التجاوب الكامل

- **تصميم يناسب جميع الأجهزة**: من الهواتف الذكية إلى أجهزة سطح المكتب
- **نقاط كسر (Breakpoints) متعددة**: تحسين التجربة عند جميع أحجام الشاشة
- **قوائم ملاحة ذكية**: قائمة هامبرجر على الأجهزة الصغيرة

### ✨ التأثيرات والانتقالات

- **رسوم متحركة سلسة**: تأثيرات fadeIn و slideUp عند تحميل الصفحة
- **انتقالات على التمرير**: تأثيرات hover جذابة على البطاقات والأزرار
- **تحول ديناميكي**: عناصر متحركة عند التفاعل

### 🔧 البنية المتحسنة

- **أقسام جديدة**:
  - About (عن المؤسسة) - مع معلومات مفصلة
  - Programs (الأقسام والبرامج) - عرض الأقسام المختلفة
  - Stats (الإحصائيات) - أرقام مهمة عن المدرسة
  - Contact (تواصل معنا) - نموذج اتصال كامل
  - Footer (التذييل) - روابط سريعة ومعلومات التواصل

### 📝 نموذج الاتصال

- حقول ذكية مع التحقق الأساسي
- واجهة سهلة الاستخدام
- معلومات الاتصال المباشرة

### 🌐 نظام الألوان المتقدم

```
الأساسي: #0b5fab (أزرق)
الثانوي: #f39c12 (برتقالي)
النجاح: #10b981 (أخضر)
التحذير: #f59e0b (أصفر)
الخطأ: #ef4444 (أحمر)
```

## الملفات

- `index.html` - الملف الرئيسي (محدث بالكامل)
- `style.css` - ورقة الأنماط الحديثة والمتجاوبة
- `assets/` - الصور والخطوط والموارد
  - `fonts/` - خطوط عربية احترافية (Cairo, Kufi, Ruqaa)
  - `images/` - صور المدرسة

## التكنولوجيا المستخدمة

- HTML5 (معايير حديثة)
- CSS3 (مع متغيرات CSS والشبكات المتقدمة)
- JavaScript (للتنقل على الجوال)
- Flexbox و Grid - للتخطيط الحديث
- RTL (من اليمين إلى اليسار) - للنصوص العربية

## نقاط الاستجابة (Breakpoints)

- **Desktop**: 1200px وما فوق
- **Tablet**: 768px - 1199px
- **Mobile**: أقل من 480px

## ميزات الأداء

- ✅ تحميل سريع للخطوط
- ✅ صور محسنة
- ✅ CSS منظم وفعال
- ✅ بدون إطارات عمل خارجية (vanilla CSS)

## كيفية التخصيص

### تغيير الألوان

قم بتعديل المتغيرات في `:root` في ملف `style.css`:

```css
--primary: #0b5fab; /* لون أساسي */
--secondary: #f39c12; /* لون ثانوي */
```

### تغيير الخطوط

المتغيرات المتاحة:

- `--font-family-kufi` - Kufi (الخط الأساسي)
- `--font-family-cairo` - Cairo
- `--font-family-ruqaa` - Ruqaa

### تعديل المسافات

استخدم متغيرات المسافة:

```css
--spacing-md: 1rem;
--spacing-lg: 1.5rem;
--spacing-xl: 2rem;
```
